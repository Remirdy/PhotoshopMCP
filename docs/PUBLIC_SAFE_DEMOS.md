# Public-Safe Demo Policy

## Statement

All demo content, example workflows, layer names, design specs, and export templates included in Remirdy Photoshop MCP are **fully fictional and created for demonstration purposes only**.

## What Is Included

| Content | Status |
|---------|--------|
| "Color Crate Sort" game concept | ✅ Fictional — invented for this demo |
| Layer names and group structure | ✅ Generic professional conventions |
| Design tokens and color palettes | ✅ Original — not based on any real product |
| Export file names | ✅ Derived from fictional project name |
| Workflow presets | ✅ Generic mobile game UI conventions |

## What Is NOT Included

| Content | Status |
|---------|--------|
| Real client names or briefs | ❌ Not present |
| Real studio case studies | ❌ Not present |
| Private art tests | ❌ Not present |
| Confidential design specs | ❌ Not present |
| Any proprietary game mechanics | ❌ Not present |

## Why This Matters

This repository is intended to be published as **public open-source**. The demo content must be safe for GitHub, npm, and any public distribution channel.

If you are adding new examples or workflows to this repository:
1. Use fictional project names
2. Use generic, non-proprietary layer conventions
3. Do not reference real clients, studios, or products
4. Do not include assets with unknown licensing

## Fictional Demo: Color Crate Sort

"Color Crate Sort" is a **made-up casual puzzle game**. It is used purely to demonstrate what a premium mobile game UI PSD structure looks like. There is no game called "Color Crate Sort" — it was invented to give the demo a concrete, relatable name without referencing any real product.
